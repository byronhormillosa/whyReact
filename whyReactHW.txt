#### Exercises
>In your own words, explain single-page applications and progressive web apps.

A: 

Single-page web applications interact with the user by dynamically re-writing the current page instead of navigating and loading to a different page from a server. This method makes the user experience seamless, gives the feel of a desktop application.

Progressive web apps combine the web and mobile application experience. These require no installation and can be used in a browser tab. Once a user finds it, the web app can be immediately be used. It is designed to be responsive, meaning it can be compatible with different devices.
